Ce plugin gère un livre d'or basé sur un code (unique) à entrer. L'utilisateur doit entrer un code valide lorsqu'il veut enregistrer un message sur le livre d'or.
Du côté du panneau d'administration, il est possible de générer ces codes mais aussi de modérer les messages (insultes, etc.).
